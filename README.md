# DecodeLabs-Internship--Week-3
This is for Week - 3 Project
A simple Random Password Generator is built using Python and Tkinter. It's used to generate Strong, Customaizable passwords without taking much of time.

**Software Used :-**
1. Python

2. Tkinter

How it works:

1. Pick your desired password length using a slider
2. Toggle symbols and digits on or off
3. Hit generate and copy your new password
